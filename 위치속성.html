Position

Static Position

static position div tag
static position div tag

position을 설정해주지 않았을 때 기본 값
이전 요소가 block 이라면 다음 줄에 위치
이전 요소가 inline 이라면 옆에 위치

Relative Position

relative position div tag

relative만 사용 시 static과 동일
left/right/top/bottom을 이용해 위치 조정 가능
현재 위치에서 상대적으로 이동

Fixed Position

fixed position div tag

fixed는 브라우저 화면상에서 고정
left/right/top/bottom을 이용해 위치 조정 가능

Absolute Position

absolute position div tag
absolute position div tag
relative position div tag

absolute는 화면상에 절대값으로 출력되는 위치
bottom은 값이 달라지기 때문에, top/right 위주로 위치를 지정
다른 태그안에 들어가 있어도 위치가 바뀌지 않는다 relative는 static의 위치에서 이동하기 때문에
다른 태그에 있을 때, 그 태그의 위치에 따라 relative의 위치가 바뀐다
